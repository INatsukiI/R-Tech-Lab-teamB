## 使用した技術
- javascript(node.js),html,css(bootstrap)  
- socket.io(リアルタイム通信)  
- jQuery  
- sqlite3(データベース)  

## アプリの起動

1. 次のコマンドを実行します

    ```bash
    # アプリのディレクトリに移動
    cd ~/chatapp
    # アプリの開発で使用するライブラリをインストール（初回のみ）
    npm install
    # 起動コマンドを実行
    npm start
    ```

2. `http://サーバのIPアドレス:ポート番号/` にブラウザでアクセスします

    例：`http://127.0.0.1:3000/`
